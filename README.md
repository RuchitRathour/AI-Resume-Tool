# AI Resume Tool

An AI-powered Resume Generator built using Java, Spring Boot, Spring AI, React, and REST APIs. It helps users generate professional resumes quickly with AI assistance.

## 🚀 Live Demo

🔗 ai-resume-ruchit.netlify.app

## ✨ Features

* AI-powered resume generation
* Responsive and user-friendly UI
* REST API integration
* React frontend with Vite
* Spring Boot backend
* Download professional resumes

## 🛠 Tech Stack

### Backend

* Java
* Spring Boot
* Spring AI
* Maven

### Frontend

* React
* Vite
* Tailwind CSS

## 📂 Project Structure

```
AI-Resume-Tool
├── resume-ai-backend
└── resume_frontend
```

## ⚙️ Installation

### Backend

```bash
cd resume-ai-backend
mvn spring-boot:run
```

### Frontend

```bash
cd resume_frontend
npm install
npm run dev
```
